Add your Mapbox access token

The map and routing need a Mapbox token. Create a free one at account.mapbox.com (a public pk.… token), then add it in Vercel → Settings → Environment Variables:

MAPBOX_TOKEN = pk.your_token_here

Redeploy after setting it. Tip: restrict the token to your site's URL in the Mapbox dashboard so it can't be reused elsewhere. (Locally, run the dev server with MAPBOX_TOKEN=… node scripts/dev-server.js.)
